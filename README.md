# Sales Performance Dashboard – Power BI

An interactive Power BI dashboard designed to analyze sales performance, profitability, order trends, and regional insights to support data-driven business decisions.

## Objective
To analyze sales data and identify trends, high-performing regions, and profitable product categories using interactive dashboards.

## Dataset
- sales_raw.csv – original dataset
- sales_cleaned.xlsx – cleaned and transformed data used for analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Monthly Sales Trend

## Dashboard Features
- KPI cards for Sales, Profit, Profit Margin, and Orders
- Monthly and yearly sales trend analysis
- Region-wise and category-wise performance analysis
- Interactive slicers for dynamic filtering

## Outcome
This project demonstrates my ability to clean raw data, build DAX measures, and design interactive dashboards that convert data into actionable business insights.
